# План внедрения системы контекстных подсказок (Help System)

Вместо сложного линейного онбординга мы внедрим систему постоянных контекстных подсказок. Рядом с важными функциями, кнопками и заголовками появится иконка `?`, которая при клике будет открывать лаконичное описание функционала.

## 1. Структура данных (i18n)

Будет создан новый namespace `help` с файлами `help.json` для каждого языка.

### Ключевые топики:
- **Дашборд**: заголовок, вкладки (Анализ резюме, Кандидаты, Вакансии).
- **Вакансии**: создание вакансии, идеальный профиль, управление.
- **Кандидаты**: приглашение, база, статусы тестов.
- **Массовый анализ**: загрузка, выбор вакансий, вердикты.
- **Воронка (Funnel)**:
    - Автоматические переходы (Testing, Evaluated).
    - AI-инструментарий (План интервью, Сравнение кандидатов).
    - Управление статусами (Hired, Rejected).

## 2. Дизайн компонента `HelpCircle`

Компонент будет реализован в `src/shared/ui/HelpCircle.tsx`:
- **Иконка**: `HelpCircle` из `lucide-react` (размер 14-16px, цвет `muted-foreground`).
- **Механика**: `Popover` из `shadcn/ui` (открытие по клику).
- **Стиль**: Glassmorphism (матовое стекло) для содержимого.
- **Пропсы**: `topicId` (ключ из `help.json`).

## 3. Точки интеграции (Где появятся вопросы)

### Главный дашборд:
- У заголовка "HR Панель".
- У каждой вкладки (Resume Analysis, Candidates, Vacancies).

### Вкладка "Вакансии":
- Кнопка "Создать вакансию".
- Блок "Кандидаты" на карточке вакансии.
- Кнопка "Управление".

### Редактор вакансии / Идеальный профиль:
- Заголовок "Идеальный психометрический профиль".
- Кнопка "Сгенерировать идеальный профиль".

### Вкладка "Кандидаты":
- Кнопка "Сгенерировать ссылку".
- Индикаторы прогресса тестов на карточке.

### Воронка (Funnel):
- Заголовки колонок (особенно Testing, Evaluated, Interview).
- Кнопки генерации анализа и приглашения на интервью внутри карточек.
- Кнопка "Сравнить кандидатов".

## 4. Этапы реализации (для Code Mode)

1. **Локализация**: Создание `public/locales/{ru,en,kk}/help.json` и регистрация namespace в `i18n.ts`.
2. **Компонент**: Реализация `HelpCircle.tsx` с использованием `Popover`.
3. **Интеграция (Дашборд)**: Добавление подсказок на главную страницу и вкладки.
4. **Интеграция (Фичи)**: Последовательное добавление в `VacancyList`, `CandidateList`, `ResumeAnalysis`.
5. **Интеграция (Воронка)**: Самая детальная часть — добавление подсказок в `VacancyFunnel` и `CandidateCard`.
6. **Полировка**: Проверка адаптивности на мобильных (Popover должен корректно центрироваться).
