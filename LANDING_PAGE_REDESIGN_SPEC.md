# Техническое задание: Редизайн Landing Page (HR Platform v2.0)

**Дата:** 14 января 2026
**Статус:** Draft
**Цель:** Создать современную, продающую главную страницу уровня топ-SaaS компаний (Linear, Stripe, Raycast), которая эффективно презентует возможности платформы, подчеркивает AI-функционал и прозрачную модель оплаты.

---

## 1. Концепция и Дизайн

### 1.1 Визуальный стиль
*   **Стиль:** "Clean & Modern SaaS". Много воздуха, крупная типографика, скругленные углы (`rounded-3xl`), эффекты матового стекла (Glassmorphism), мягкие градиенты в фирменных цветах (Violet/Indigo).
*   **Анимации:** Плавное появление элементов при скролле (`framer-motion`), микро-взаимодействия при наведении (hover effects).
*   **Адаптивность:** Полная поддержка мобильных устройств (Mobile First). Скриншоты должны быть читаемыми или адаптированными.
*   **Тон (Tone of Voice):** Профессиональный, но не скучный. Уверенный, технологичный, заботливый.

### 1.2 Ключевые сообщения
1.  **AI-First:** Искусственный интеллект — это ядро платформы, а не просто фича.
2.  **No Subscriptions:** Платите только за результат (токены). Свобода от месячных платежей за "воздух".
3.  **Smart Matching:** Объективная оценка совместимости (Tests + Hard Skills + Soft Skills).
4.  **All-in-One:** Весь цикл найма в одном окне.

## 2. Структура Страницы (Секции)

Страница будет состоять из следующих последовательных блоков:

### 2.1 Hero Section (Первый экран)
*   **Заголовок:** "Найм нового поколения, усиленный искусственным интеллектом".
*   **Подзаголовок:** "Автоматизируйте рутину, исключите предвзятость и находите идеальных кандидатов быстрее с помощью HR Platform v2.0".

### 2.2 Features Grid (Экосистема AI)
*   **Заголовок:** "Платформа, пронизанная интеллектом".
*   **Описание:** "AI-ассистент сопровождает вас на каждом этапе: от анализа резюме до оффера."

    1.  **Анализ резюме:** "Мгновенный разбор сотен PDF". (Иконка: FileText).
    2.  **Генерация документов:** "Офферы и отказы за секунды". (Иконка: FileCheck).
    3.  **AI-Ассистент:** "Умный чат-бот, знающий контекст вашей компании". **Скриншот №8**.
    4.  **Психометрия:** "Глубокий анализ личности (Big5, MBTI, DISC)". **Скриншот №7**.

### 2.3 Talent Market (Умный подбор)
*   **Заголовок:** "Рынок талантов с Match Score".
*   **Описание:** "Не просто база резюме. Наша система автоматически рассчитывает процент соответствия кандидата вашей вакансии, учитывая Hard Skills, Soft Skills и результаты тестов."
*   **Визуал:** **Скриншот №2 (Talent Market)** или **Скриншот №4 (Profile)** с акцентом на графики совместимости.
*   **Лейаут:** Текст слева, изображение справа (на десктопе).

### 2.4 Hiring Funnel (Воронка найма)
*   **Заголовок:** "Прозрачный процесс найма".
*   **Описание:** "Канбан-доска, автоматические переходы статусов и удобное управление кандидатами."
*   **Визуал:** **Скриншот №5 (Funnel)**.
*   **Лейаут:** Текст справа, изображение слева.

### 2.5 Pricing (Модель оплаты)
*   **Заголовок:** "Свобода от подписок".
*   **Описание:** "Забудьте о ежемесячных списаниях. Вы платите только за конкретные действия (токены). Нет найма — нет расходов."
*   **Визуализация:** Простые карточки с примерами стоимости ("Анализ резюме = X токенов", "Покупка контакта = Y токенов"). Можно использовать **Скриншот №9 (Billing)** как фон или иллюстрацию.

### 2.6 Call to Action (Финал)
*   **Текст:** "Готовы трансформировать свой HR-отдел?"
*   **Кнопка:** Большая кнопка "Зарегистрироваться и получить 5000 токенов".

### 2.7 Footer
*   Ссылки на Privacy, Terms, Support.
*   Копирайт.

## 3. Техническая Реализация

### 3.1 Компоненты
Создать новые компоненты в `src/pages/public/landing/components/`:
*   `LandingHero.tsx`
*   `LandingFeatures.tsx`
*   `LandingShowcase.tsx` (универсальный блок "Картинка + Текст")
*   `LandingPricing.tsx`
*   `LandingFooter.tsx`
*   `LocalizedScreenshot.tsx` (см. пункт 3.2)

### 3.2 Локализация Скриншотов
Создать компонент `LocalizedScreenshot`, который автоматически выбирает нужный файл из `public/pics` в зависимости от текущего языка (`i18n.language`).

**Логика:**
*   Принимает проп `name` (например, "1").
*   Определяет суффикс: `ru` -> "ru", `kk` -> "kk", иначе -> "en".
*   Рендерит `img src="/pics/${name}${suffix}.png"`.
*   Добавляет красивую рамку (border-radius, shadow) и, возможно, мокап браузера.

**Маппинг скриншотов:**
*   **1 (Dashboard):** Hero Section.
*   **2 (Talent Market):** Секция "Рынок талантов".
*   **4 (Candidate Profile):** Альтернатива для рынка талантов или раздела "Аналитика".
*   **5 (Funnel):** Секция "Воронка".
*   **6 (Chat):** Раздел коммуникации (можно объединить с воронкой).
*   **7 (Testing):** Раздел психометрии.
*   **8 (AI Assistant):** Раздел AI-фич.
*   **9 (Billing):** Раздел цен.

### 3.3 Локализация Текстов
Добавить новые ключи в `public/locales/{ru,en,kk}/landing.json`.
Структура JSON:
```json
{
  "hero": { ... },
  "features": { ... },
  "talentMarket": { ... },
  "funnel": { ... },
  "pricing": { ... },
  "cta": { ... }
}
```

### 3.4 Анимации
Использовать `framer-motion`:
*   `initial={{ opacity: 0, y: 20 }}`
*   `whileInView={{ opacity: 1, y: 0 }}`
*   `viewport={{ once: true }}`
Для списков использовать `staggerChildren`.
